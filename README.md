# blockchain-dapp

## 如何运行

1. 在本地启动ganache应用。

2. 在 `./contracts` 中安装需要的依赖，运行如下的命令：
    ```bash
    npm install
    ```
3. 在工程目录中编译合约，运行如下的命令：
    ```bash
    truffle compile
    ```
4. 在工程目录中部署合约，运行如下的命令：
    ```bash
    truffle migrate
    ```
5. 在 `./app` 中启动前端程序，运行如下的命令：
    ```bash
    npm run dev
    ```
